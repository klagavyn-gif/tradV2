Core Flow
เริ่มจากส่วนวิเคราะห์ กรอกหลาย Symbol, เลือกช่วงเวลา, และส่งผลเข้าหน้าเดียวแบบอ่านง่าย
Monitor
ใช้ Watchlist เป็นคิวติดตาม เพิ่มหุ้นที่สนใจไว้ล่วงหน้า แล้วกดวิเคราะห์ซ้ำได้ทันทีโดยไม่ต้องพิมพ์ใหม่
Signal Feed
ดูสัญญาณเด่นแยกอีกฝั่ง รวม alert ที่สำคัญจากรายการติดตาม เพื่อกดกลับไปวิเคราะห์เชิงลึกได้เร็วขึ้น
Analysis
ค้นหาและวิเคราะห์สินทรัพย์
กรอกหลาย Symbol ได้ในครั้งเดียว แล้วดูแผนเทรด, ความเสี่ยง และกราฟในบล็อกเดียวกัน
รองรับหลาย Symbol พร้อมกัน

System
